## Interpretation

The LLM recovers structured oncology fields from free-text notes with 94.3% macro accuracy. All core staging, histology, biomarker status, ECOG, and PD-L1 fields achieve 100% accuracy — the model reads these unambiguously from the synthetic note templates.

The two weaker fields are expected: `kras_variant` (74% accuracy, 100% miss rate) reflects that the synthetic notes often state KRAS status without specifying the exact codon variant (e.g., G12C vs. G12D), leaving nothing for the model to extract. `progression_mentioned` (86%) shows occasional false positives — the model flags disease-course language in notes where progression is described retrospectively rather than newly documented.

For the list fields, the model achieves perfect recall on metastatic sites (it finds every site mentioned) but overcounts slightly (precision 64%), likely pulling historical sites from prior-history sections the ground truth does not include. Treatment recall (91.5%) is strong; the precision gap (73.5%) reflects background drug mentions that are not part of the current regimen.
